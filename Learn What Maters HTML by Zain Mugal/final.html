Complete HTML5 Mastery

1. Meta Tags & SEO Essentials

Purpose: Control page metadata & search engine behavior 
<!-- Essential Meta Tags -->
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Page description">

<!-- Open Graph (Social Media) -->
<meta property="og:title" content="Page Title">
<meta property="og:image" content="image.jpg">

2. Head Section Elements

<title>Page Title</title>
<link rel="stylesheet" href="styles.css">
<link rel="icon" type="image/x-icon" href="favicon.ico">
<script src="script.js" defer></script>
<style>
    /* In-page CSS */
</style>

3. Special Characters & Symbols

Copyright: © | Trademark: ™ | Euro: €

Arrow: → | Emoji: 😀 | Math: ∑ (∑)

&copy;   &trade;   &#128512;   &sum;

4. Embedding External Content

<iframe 
    src="https://maps.google.com/maps?q=new+york" 
    title="NYC Map" 
    loading="lazy"
></iframe>

<embed type="video/webm" src="video.mp4" width="400">

5. HTML5 Native Features

<!-- Geolocation -->
<div id="location"></div>

<!-- Local Storage -->
<script>
localStorage.setItem('key', 'value');
</script>

<!-- Canvas -->
<canvas id="myCanvas" width="200" height="100"></canvas>

6. Global Content Support

<html lang="ar" dir="rtl"> <!-- Arabic -->
<bdi>مرحبا</bdi> <!-- Bi-directional isolation -->

<ruby>
    漢 <rp>(</rp><rt>kan</rt><rp>)</rp>
</ruby>

7. Professional HTML Guidelines

Essential Rules:
  • Always validate HTML (W3C Validator)
  • Use semantic elements first
  • Keep nesting under 5 levels
  • Add alt text to all images

8. Outdated Elements to Avoid

<!-- Deprecated -->
<font face="arial">Text</font>
<center>Content</center>
<marquee>Scrolling</marquee>

<!-- Modern Alternatives -->
<div style="text-align: center">Content</div>
<span style="font-family: arial">Text</span>

9. Specialized HTML5 Tags

<details>
    <summary>Read More</summary>
    Hidden content
</details>

<dialog open>
    <p>Popup content</p>
</dialog>

<template>
    <div>Reusable content</div>
</template>

10. HTML5 Feature Checklist

Must-Know Features:
  • Semantic elements (header, nav, article)
  • Form validation attributes
  • Audio/Video embedding
  • Canvas/SVG graphics
  • Web Storage APIs
  • Responsive image (srcset)
