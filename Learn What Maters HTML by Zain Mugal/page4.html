Chapter 4: Power of Images in HTML

The Essential img Tag

<img src="image.jpg" alt="Description" width="800" height="600">
Attributes Breakdown:
  • src: Image source (required)
  • alt: Accessibility description (required)
  • width/height: Prevent layout shifts
  • loading: lazy loading control

Real-World Image Examples

E-commerce Product Image

<figure>
    <img src="product-123.jpg" 
         alt="Blue Running Shoes - Model X2023" 
         width="600" 
         height="400"
         loading="lazy">
    <figcaption>Our best-selling running shoes</figcaption>
</figure>

Responsive Hero Image

<picture>
    <source media="(min-width: 1200px)" srcset="hero-large.webp">
    <source media="(min-width: 768px)" srcset="hero-medium.jpg">
    <img src="hero-small.jpg" 
         alt="Mountain landscape" 
         style="width:100%;height:auto">
</picture>

Advanced Image Techniques

Image Optimization Checklist:
  • Use WebP format for modern browsers
  • Implement responsive images with srcset
  • Add decoding="async" for performance
  • Use <picture> for art direction

Image Map Example

<img src="workstation.jpg" alt="Office setup" usemap="#officeMap">
<map name="officeMap">
    <area shape="rect" coords="34,44,270,350" 
          href="monitor.html" 
          alt="Computer Monitor">
    <area shape="circle" coords="337,300,44" 
          href="coffee.html" 
          alt="Coffee Cup">
</map>

Accessibility Deep Dive

Critical Requirements:
  • Never omit alt attribute
  • Use empty alt="" for decorative images
  • Provide text alternatives for complex images
  • Use ARIA roles for CSS background images
