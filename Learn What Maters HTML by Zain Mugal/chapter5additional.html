Detailed Input Type Breakdown

Type Description Key Attributes Example range Slider control min, max, step <input type="range" min="0" max="100" step="5"> color Color picker value <input type="color" value="#3498db"> tel Phone number input pattern <input type="tel" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}">

Advanced Validation Patterns

<!-- Custom Validation -->
<input type="text" 
       pattern="(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,}" 
       title="Must contain at least 8 characters, including UPPER/lowercase and numbers">

Complex Form Example

<form id="registration" novalidate>
    <fieldset>
        <legend>Account Details</legend>
        
        <label>
            Username:
            <input type="text" name="username" required 
                   minlength="4" maxlength="20">
        </label>

        <label>
            Birth Date:
            <input type="date" name="dob" 
                   min="1900-01-01" max="2023-12-31">
        </label>
    </fieldset>

    <fieldset>
        <legend>Preferences</legend>
        
        <label>
            <input type="checkbox" name="newsletter" checked>
            Subscribe to newsletter
        </label>

        <label>
            Theme Color:
            <input type="color" name="theme" value="#3498db">
        </label>
    </fieldset>
</form>
