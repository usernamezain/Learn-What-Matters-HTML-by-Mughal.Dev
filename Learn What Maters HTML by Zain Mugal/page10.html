Chapter 10: Mastering HTML Lists

Why Use Lists:
  • Organize related items
  • Improve content structure
  • Enhance accessibility
  • Better SEO ranking

1. Unordered Lists (<ul>)

Features List

  • 24/7 Support
  • Free Updates
  • Secure Hosting
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

2. Ordered Lists (<ol>)

Installation Steps

  1. Download Software
  2. Run Installer
  3. Complete Setup
<ol type="I" start="5">
    <li>Step One</li>
    <li>Step Two</li>
</ol>

3. Description Lists (<dl>)

Terminology

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
<dl>
    <dt>Term</dt>
    <dd>Definition</dd>
</dl>

4. Nested Lists

Table of Contents

  1. Introduction
    • Overview
    • History
  2. Main Content
<ol>
    <li>Chapter 1
        <ul>
            <li>Section 1.1</li>
        </ul>
    </li>
</ol>

List Attributes

Attribute Description
type Bullet/number style (1, A, a, I, i)
start Starting number
reversed Reverse numbering

Best Practices

When to Use Which List:
  • Ordered: Sequential information
  • Unordered: Grouping related items
  • Description: Terms and definitions
Avoid:
  • Using lists for layout
  • Excessive nesting (3+ levels)
  • Empty list items

Real-World Example

  • Home
  • About
  • Contact
<nav>
    <ul>
        <li><a href="/">Home</a></li>
        <li><a href="/about">About</a></li>
    </ul>
</nav>
