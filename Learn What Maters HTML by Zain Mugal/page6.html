Chapter 6: Mastering HTML Attributes

Core Attributes

Attribute Description Example
id Unique identifier <div id="header">
class Space-separated class names <p class="text warning">
title Advisory information <abbr title="World Health Organization">WHO</abbr>
style Inline CSS (use sparingly) <span style="color:red">
Best Practices:
  • Use IDs for unique elements only
  • Follow BEM naming convention for classes
  • Avoid inline styles when possible

Data Attributes

<div data-user-id="123" 
     data-role="admin" 
     data-filter-type="urgent">
</div>
Access custom data in JavaScript using: element.dataset.userId

ARIA Attributes

<nav aria-label="Main navigation">
    <ul role="menubar">
        <li role="menuitem">Home</li>
    </ul>
</nav>

Event Handler Attributes

Attribute Description Example
onclick Mouse click handler <button onclick="submitForm()">
onload Element loaded <body onload="init()">
onchange Value changed <input onchange="validate()">

Custom Attributes

<!-- Valid in HTML5 -->
<div custom-attribute="value">

<!-- XHTML requires data- prefix -->
<div data-custom-attribute="value">

Boolean Attributes

<input disabled>
<video controls>
<option selected>
<details open>
Important: Boolean attributes don't need values. <input disabled="disabled"> is also valid but redundant.

Global Attributes

<div contenteditable="true" 
     spellcheck="false" 
     hidden
     lang="en">
</div>

Real-World Attribute Usage

<form id="login-form" 
      class="auth-form dark-theme" 
      novalidate
      data-submit-url="/api/login">

    <input type="email" 
           id="email-input" 
           name="email" 
           required
           aria-describedby="email-help">

    <div id="email-help" 
         class="helper-text" 
         role="alert">
        Enter valid email address
    </div>
</form>
