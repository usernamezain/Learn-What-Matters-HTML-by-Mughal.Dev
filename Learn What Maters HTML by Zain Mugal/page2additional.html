Presentational Tags (Use CSS Instead)

Warning: These are presentational and should be styled with CSS
  • <b> → Use <strong> or CSS font-weight
  • <i> → Use <em> or CSS font-style
  • <u> → Use CSS text-decoration
  • <tt>, <big>, <center> → Obsolete
