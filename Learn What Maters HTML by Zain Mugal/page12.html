SVG: Scalable Vector Graphics

Why SVG Matters:
  • Resolution-independent graphics
  • Smaller file sizes for complex shapes
  • Accessible through DOM manipulation
  • Animatable and styleable with CSS
  • Essential for data visualization

1. Key Features

When to Use SVG:
  • Logos & icons
  • Charts & graphs
  • Interactive maps
  • Complex animations
  • Resolution-sensitive images

2. Basic Shapes

Circle

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" 
            fill="#3498db" stroke="#2c3e50" stroke-width="2"/>
</svg>

3. Path Element (Complex Shapes)

<svg width="200" height="200">
    <path d="M10 80 C 40 10, 65 10, 95 80 S 150 150, 180 80" 
          stroke="#3498db" fill="none" stroke-width="2"/>
</svg>

4. Text in SVG

<svg width="300" height="100">
    <text x="20" y="35" font-family="Arial" font-size="20"
          fill="#2c3e50" transform="rotate(30 20,40)">
        Scalable Text!
    </text>
</svg>

5. SVG Animation

<svg width="100" height="100">
    <rect width="50" height="50" fill="#3498db">
        <animate attributeName="x" from="0" to="50" 
                 dur="2s" repeatCount="indefinite"/>
    </rect>
</svg>

6. SVG vs PNG Comparison

Feature SVG PNG
Scalability ✅ Infinite ❌ Pixelates
File Size Small (simple) Large (high-res)

7. Best Practices

Pro Tips:
  • Optimize SVG files with tools like SVGO
  • Use <symbol> for reusable components
  • Prefer inline SVG for interactivity
  • Add aria labels for accessibility
