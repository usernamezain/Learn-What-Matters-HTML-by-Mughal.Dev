Table Tag Anatomy

<table>
Container element for all table components
<tr>
Table row - Horizontal group of cells
<td>
Table data cell - Contains actual content
<th>
Table header cell - Describes column/row
<thead> <tbody> <tfoot>
Semantic grouping elements

Accessibility Essentials

scope attribute
Identifies header direction: col|row|colgroup|rowgroup
headers attribute
Links cells to multiple headers: headers="header1 header2"
caption element
Provides table context for screen readers
<th scope="col" id="price">Price</th>
<td headers="price">$19.99</td>
