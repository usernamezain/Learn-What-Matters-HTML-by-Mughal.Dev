Chapter 11: HTML Tables Deep Dive

Definition: HTML tables arrange data into rows and columns of cells Purpose: Display tabular data like prices, schedules, or statistics

When to Use Tables

  • Financial reports
  • Comparison charts
  • Timetables/schedules
  • Data grids

Basic Table Structure

<table>
    <caption>Monthly Sales</caption>
    <thead>
        <tr>
            <th>Month</th>
            <th>Sales</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>January</td>
            <td>$5000</td>
        </tr>
    </tbody>
</table>

Table Tags Definitions

Tag Purpose
<table> Wrapper element for all table content
<caption> Table title/description
<thead> Header content container
<tbody> Main table content container
<tfoot> Footer content container
<tr> Table row container
<th> Header cell
<td> Standard data cell

Key Attributes

<td colspan="2">Span 2 columns</td>
<td rowspan="3">Span 3 rows</td>
<th scope="col">Column header</th>

Best Practices

Do:
  • Use <th> for headers
  • Add <caption> for context
  • Use scope attribute
Don't:
  • Use tables for page layout
  • Skip semantic markup
  • Nest tables excessively

Accessibility Features

<table aria-describedby="summary">
    <caption id="summary">Sales Data</caption>
    <th scope="col">Month</th>
</table>
