Chapter 2: HTML Text Formatting

Headings & Paragraphs

HTML provides six levels of headings and paragraph tags for text organization:

<h1>Main Title</h1>
<h2>Subheading</h2>
<h3>Section Heading</h3>
<p>This is a paragraph of text.</p>
Best Practice: Use heading tags in order (h1 → h6) for accessibility and SEO.

Text Formatting Tags

Essential tags for emphasizing content:

<strong> makes text bold and important
<em> adds italic emphasis
<mark> highlights text
<small> creates smaller text
<del> shows deleted text
<ins> indicates inserted text
<sub> subscript2
<sup> superscript3 

<p>
    <strong>Important:</strong> This is <em>emphasized text</em>.
    <mark>Highlighted content</mark> and H<sub>2</sub>O.
</p>

Line Breaks & Horizontal Rules

Control content flow with these self-closing tags:

<p>First line<br>Second line</p>
<hr>
<p>Section divider above</p>
Remember: <br> should only be used for line breaks that are actually part of the content, not for styling.

Preformatted Text

Use <pre> tag to preserve whitespace:

<pre>
    This text
        preserves
            spaces
    and line breaks
</pre>
