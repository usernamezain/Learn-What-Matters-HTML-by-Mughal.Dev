Complete List Tags Reference

<ul> - Unordered List

Definition: Creates a bullet-point list of items where order doesn't matter

Usage: For grouping related items without sequence importance

<ul>
    <li>First item</li>
    <li>Second item</li>
</ul>

<ol> - Ordered List

Definition: Creates a numbered list where item order is important

Attributes:

  • type: 1 (numbers), A (uppercase), a (lowercase), I (roman upper), i (roman lower)
  • start: Starting number
  • reversed: Reverse numbering

<ol type="A" start="3">
    <li>Third item</li>
    <li>Fourth item</li>
</ol>

<li> - List Item

Definition: Represents a single item in ordered/unordered lists

Usage Notes:

  • Can contain other HTML elements
  • Can be nested within other lists

<li>
    <h3>List Item Heading</h3>
    <p>Item description</p>
</li>

<dl> - Description List

Definition: Contains term-description pairs (name-value groups)

Child Elements:

  • <dt>: Term being described
  • <dd>: Description/definition

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
</dl>

<dt> - Description Term

Definition: Specifies a term in a description list

Rules:

  • Must be followed by <dd>
  • Multiple <dt> can precede single <dd>

<dd> - Description Details

Definition: Provides description/definition for preceding <dt>

Usage: Can contain paragraphs, images, even other lists

<dd>
    <p>Primary language for web content structure</p>
    <img src="html5-logo.png" alt="HTML5 logo">
</dd>

List Type Comparison

<ul>
Bulleted items, no order importance
<ol>
Numbered/lettered items, sequential order
<dl>
Term-definition pairs, dictionary-style

Real-World Applications

Common Use Cases:
Navigation Menus
<ul> with <li> items
Legal Documents
<ol> for numbered clauses
FAQs
<dl> for questions/answers
