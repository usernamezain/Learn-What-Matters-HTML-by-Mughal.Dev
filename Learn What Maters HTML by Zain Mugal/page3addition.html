Link Attributes Deep Dive

The rel Attribute

Control relationship between documents:

<a href="https://external.com" rel="nofollow noopener">External Site</a>
<a href="/about" rel="prev">Previous Page</a>
<a href="/contact" rel="next">Next Page</a>
Key rel values:
  • nofollow: Tell search engines not to follow
  • noopener: Prevent security vulnerabilities
  • noreferrer: Hide referral information

Fragment Identifiers

Link to specific page sections:

<!-- Create anchor point -->
<h2 id="section2">Section 2</h2>

<!-- Link to anchor -->
<a href="#section2">Jump to Section 2</a>

Accessibility Considerations

Important:
  • Always use descriptive link text
  • Add ARIA labels for complex navigation
  • Ensure keyboard navigability
  • Provide skip navigation links
<!-- Skip link for accessibility -->
<a href="#maincontent" class="skip-link">Skip to main content</a>

<!-- ARIA example -->
<nav aria-label="Main navigation">
    <!-- navigation items -->
</nav>

SEO Best Practices

  • Use descriptive anchor text
  • Maintain logical link hierarchy
  • Limit external links with nofollow
  • Use canonical links when needed
