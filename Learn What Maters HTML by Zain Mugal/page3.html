Chapter 3: Links & Navigation

Anchor Tag Basics

The <a> tag creates hyperlinks using the href attribute:

<a href="https://example.com">Visit Example</a>
Pro Tip: Always include meaningful link text for accessibility.

Link Types

External link: Open Example Site

Email link: Email Us

File download: Download PDF 

<!-- External link -->
<a href="https://example.com" target="_blank" rel="noopener">...</a>

<!-- Email link -->
<a href="mailto:contact@example.com">...</a>

<!-- File download -->
<a href="file.pdf" download>...</a>

Navigation Menus

Create semantic navigation using <nav>:

Home About Contact 
<nav>
    <a href="/">Home</a>
    <a href="/about">About</a>
    <a href="/contact">Contact</a>
</nav>

Link Best Practices

  • Use target="_blank" with rel="noopener" for security
  • Add download attribute for file downloads
  • Use relative paths for internal links (href="/about")
