Chapter 8: HTML Multimedia & Embedding

What You'll Learn

  • Embed audio/video with <audio> and <video>
  • Include external content using <iframe>
  • Create graphics with <canvas>
  • Implement responsive embeds

Audio Content

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Your browser does not support audio element
</audio>

Video Embedding

<video width="640" height="360" controls poster="preview.jpg">
    <source src="video.mp4" type="video/mp4">
    <track src="subtitles_en.vtt" kind="subtitles" 
           srclang="en" label="English">
    Your browser does not support video tag
</video>
Best Practices:
  • Always include controls attribute
  • Provide multiple source formats
  • Use poster for preview image

Iframes for External Content

<iframe src="https://www.example.com" 
        title="Example Site"
        width="100%" 
        height="500"
        loading="lazy">
</iframe>

Canvas Element

<canvas id="myCanvas" width="200" height="100">
    Fallback text for non-supporting browsers
</canvas>

<script>
    const canvas = document.getElementById('myCanvas');
    const ctx = canvas.getContext('2d');
    ctx.fillStyle = '#3498db';
    ctx.fillRect(10, 10, 150, 80);
</script>

Embedding Best Practices

Security Considerations:
  • Use sandbox attribute in iframes
  • Avoid cross-origin security issues
  • Implement Content Security Policy (CSP)
