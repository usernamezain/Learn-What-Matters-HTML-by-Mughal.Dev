Chapter 1: Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the foundation of every webpage on the internet. It's not a programming language, but rather a markup language used to structure content on the web.

Basic HTML Structure

Every HTML document follows this fundamental structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <!-- Page content -->
</body>
</html>

Key Components

  • <!DOCTYPE html> - Declares HTML5 document type
  • <html> - Root element of the page
  • <head> - Contains meta information
  • <body> - Contains visible content

Your First Web Page

Let's create a simple page that displays "Hello World":

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>Welcome to my website</p>
</body>
</html>
